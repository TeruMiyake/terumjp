アプリ利用前の準備

このページ上のアプリケーションを実行するためには、以下のページから証明書のインストールをして頂くことが必要となります。

証明書をインストールすることで、私（テル）により署名/発行されたアプリケーションを信頼する（≒インストールを許可する）ことになります。

自己署名証明書

インストール方法

1. 上記リンクをクリックして、証明書ファイルをダウンロード

2. ダウンロードしたファイルを開く

3. "証明書のインストール" をクリック

4. "ローカルコンピューター" を選択して OK

4. "すべて次のストアに配置する" → "信頼されたルート証明機関" を選択して OK
